Approach
Curriculum includes two six-hour interactive courses taught by experts from GitHub.

For those new to Gamma and those new to Git and GitHub are encouraged to complete both courses (Fundamentals and Advanced) to upskill their Git and GitHub knowledge. For those with fundamental knowledge of Git and GitHub, should look to attend the more advanced course.

See Gamma Git and GitHub Training Course Details below for information specific to course content and the targeted audience.
 
Course Enrollment
Courses are conducted in a virtual learning environment. Each course is six hours in length and runs over two consecutive days (three hours each day).

To enroll
  1. Choose the course you plan to attend (Fundamentals and/or Advanced), paying attention to the course prerequisites.
  2. Choose the course dates and time that work best for your schedule (making sure to check with your case team and Staffing (if not assigned).
  3. Click on the specific class link to complete your enrollment.
  4. Expect to receive a LAB notice indicating you have successfully registered for the course.
  5. Once the class list is confirmed, expect to receive an Outlook meeting invite with specific information regarding the course.
  6. For the Advanced course, you will not be confirmed until prerequisites have been confirmed/met (i.e., completion of the pre-class Assessment or enrollment/participation in the Fundamentals class).

Enrollment Best Practices

Consider your current workload and determine what date(s) and times work best. Always check with your case team in advance or Staffing if currently unassigned.

For those attending the Fundamentals and Advanced courses, you are encouraged to attend within a few months (to help with retention of concepts between the two courses).
 
Fundamentals Course Details
Launch Course: Fundamentals Git and GitHub Training
Courses are conducted in a virtual learning environment. Each course is six hours in length and runs over two consecutive days (three hours each day).

Description

You will learn how GitHub fits in your workflow as you make changes to projects. This course covers getting started with GitHub repositories, using the GitHub Flow to make changes to a repository locally, how to share your changes, and the preferred method of reviewing changes before ultimately merging them into the "production" ready branch.

Audience
  • Those new/newer to BCG / Gamma.
  • Those new/newer to Git and GitHub.

Prerequisites

New Hires: Best to attend the Git and GitHub Training AFTER attending the Gamma New Hire Orientation program (i.e., the NHO program includes an introduction to Git and GitHub).

Topics Covered

An introduction to GitHub:
First look • What is Git? • Understand GitHub • GitHub Notifications • GitHub Issues • Using Issues Effectively

Streamlining your workflow:
Workflow • BCG Gamma Workflow • GitHub projects

Working on a shared repository:
Setup SSH • Local config • Local clone • Where to start?

Collaborating on GitHub:
Push your changes • Create a Pull Request • Pull Request Tools • Review a Pull Request • Merge a Pull Request

Keeping up to date:
Update local • Using Git Log • Git Aliases • Other helpful aliases

 
Advanced Course Details
Launch Course: Advanced Git and GitHub Training
Courses are conducted in a virtual learning environment. Each course is six hours in length and runs over two consecutive days (three hours each day).

Description

You will utilize the knowledge gained from the Fundamentals Git and GitHub training to mitigate merge conflicts, learn advanced commands to fix mistakes you may have made, and other GitHub skills that will empower you to take on any challenge you may face.

Audience
  • Those who have used Git and GitHub in their everyday work and are comfortable using on your own.
  • Those looking to take Git and GitHub to the next level and be more effective using Git and GitHub across a team.

Prerequisites

Completion of the Fundamentals Git and GitHub Training course or successful completion of the Git and GitHub Training Assessment (see below).

Topics Covered

Workflows:
Git Flow • Merge strategies • Workflow discussion

Using GitHub with Jupyter Notebook:
GitHub Flow with Jupyter Notebook • Handling difficult merges with Jupyter Notebook

Merge conflicts and visualizing Git:
What is a merge conflict? • Resolve a merge conflict • Graphical 3-way diff & merge tools and strategies

Working with Tags and Releases:
Releases on GitHub • Moving files with Git • Staging changes, not just files • Local Tags

Fixing and Rewriting history:
Creating a local repository • Git Commit Amend • Git Revert • Git Reset • Git Reflog • Git Cherry Pick • Git Rebase

Selecting and integrating applications :
Understanding CI • Integrating with Trello • Integrating with JIRA

Setting up a successful repository:
Importing a template repository • Ignoring unimportant files • Other useful files for collaboration • Useful files for collaboration • Protected Branches and CODEOWNERS • Managing a repository • Contribution metrics • Managing projects on GitHub
 
Git and GitHub Assessment Instructions
Objective
The Git and GitHub Assessment serves as a brief "test" to ensure you have the fundamental knowledge and skills on Git and GitHub, so that you can successfully participate in the Advanced Git and GitHub training. If at any point during the Assessment, you are unable to complete the "task," you may want to consider attending the Fundamentals Git and GitHub training course.

Instructions
The Assessment resides in the Gamma GitHub Enterprise. Use the link below to start the Assessment. The Assessment will have you complete tasks to demonstrate a fundamental understanding of Git and GitHub - addressing the topics covered at the Fundamentals course. You will be asked to create comments, create a branch, clone the repository to your local machine, and create a pull request.

Assessment Link
https://github.gamma.bcg.com/pages/Gamma-GitHub-Training/BCG-Gamma/#/00-fundamentals-assessment

Trouble Accessing the Assessment
Contact the GammaPlatform team to gain access to the Gamma GitHub Enterprise and Gamma Training Organization, if you are unable to link to the Assessment.
 
Enrollment FAQs
Which course should I attend?
Read the course descriptions to determine which course works best for your Git and GitHub needs.

To attend the Advanced course you will need to demonstrate fundamental knowledge through completion of the Fundamentals course or passing the GitHub assessment.

Are there prerequisites?
Yes. If you are a new hire to BCG / Gamma you are encouraged to attend the New Hire Orientation/Onboarding program first.

For the Advanced course, we require completion of the Fundamentals course or completion of the GitHub Assessment to ensure learners have a fundamental understanding of Git and GitHub.

Will you record sessions?
Sessions will not be recorded; due to the interactive nature of the sessions and the activities, participants need to attend live.

What is the cancellation policy?
Cancellations less than 2 days prior will incur a $250 usd cancellation fee.

How close together should I attend the Fundamentals and Advanced courses?
We recognize that your work schedule will often drive when you are able to attend. If possible, it is best to attend within a month (for retention purposes), and no longer than three months apart.

Are courses limited by region?
Courses are being offered at varied times, to be more friendly for each region. We encourage you to enroll in the course that is best suited for where you are currently working. It is to your discretion to choose a course during "off" hours if that works best.
