Role | Senior Product Designer

Knowledge Navigator is a BCG internal Knowledge Database and a primary entry point for Learning & Development.

The Problem

Recent research identified that learners' biggest frustration was finding relevant learning content.

Project Goals

Analyze the existing state of how learning content and resources were displayed in Knowledge Navigator, identify possible friction points/opportunities, and develop a roadmap for enhancements based on solid research.

Digging Into Development

In addition to my product design work, I took over some light development work. The LMS platform had unique restrictions, limiting us to pure HTML & CSS and not allowing media queries to target specific breakpoints. With these limitations, this project became a masterclass in using Flexbox to create modules that worked across all breakpoints.
